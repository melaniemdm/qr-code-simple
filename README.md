# qr-code-simple

## Install

```bash
npm install --save qr-code-simple
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'qr-code-simple'
import 'qr-code-simple/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
